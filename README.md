
## Available Scripts

You need to add PRIVATE KEY and fee addresses in .env file (.env sample)

In the project directory, you can run:

### `npx hardhat compile`

compile contract files

### `npx hardhat test`

### `npm run deploy_btc`

contract deploy to btctestnet
### `npm run deploy_btc_test`

contract deploy to btc chain
